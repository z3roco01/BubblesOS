# BubblesOS
A toy operating system

## Requirements
i686-elf binutils and gcc
mtools
make
nasm
qemu-system-x86_64

## Features (implemented and planned)
- [X] FAT12
- [X] ATA driver
- [ ] VFS
- [ ] Proper keyboard driver
- [ ] ELF executer
- [ ] Memory allocater
- [ ] Real graphics(not VGA text mode)
