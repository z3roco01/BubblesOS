# BubblesOS
A toy operating system

## Requirements
i686-elf binutils and gcc </br>
mtools </br>
make </br>
nasm </br>
qemu-system-x86_64 </br>

## Features (implemented and planned)
- [X] FAT12
- [X] ATA driver
- [ ] VFS
- [ ] Proper keyboard driver
- [ ] ELF executer
- [ ] Memory allocater
- [ ] Real graphics(not VGA text mode)
